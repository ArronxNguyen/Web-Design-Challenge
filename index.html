Project Summary







With this project we've sort to answer the question of "What's the weather like as we approach the equator?". And use it to potentially assess some good locations for a holiday given the time of the year. The project was achieved by creating a Python script to visualize the weather of 500+ cities across the world of varying distance from the equator. To accomplish this, we used a simple Python library, the OpenWeatherMap API, and a little common sense to create a representative model of weather across world cities.
